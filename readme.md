## Get it up and running
1. Install Node.js (https://nodejs.org/)
2. Install Cordova (https://cordova.apache.org/docs/en/4.0.0/guide_cli_index.md.html)
3. Install Ionic (http://ionicframework.com/getting-started/)
4. Run: `ionic serve`
